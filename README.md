# 🌸 Hanni Pham — Interactive Fanpage

A personal fanpage dedicated to Hanni from NewJeans, built with pure vanilla JavaScript, HTML, and CSS.  
This project is a reflection of my appreciation for her personality and artistry.

---

## ✨ Features

- 🎨 **6 Color Themes** — Soft Pink, Lavender, Mint, Peach, Sky, Sunset  
- 🌙 **Dark Mode** & 💕 **Cozy Mode** — switch between elegant and warm vibes  
- 🎵 **Music Player** — play Hanni's cover of *Darling* with a visualizer  
- 📸 **Photo Booth** — add frames, filters, and drag-and-drop stickers  
- 🐰 **Bunny Hop Game** — a cute endless runner game with high score saved locally  
- 🎵 **Personal Playlist** — a curated list of songs that match Hanni's vibe  
- 🧹 **Sticker Deco** — click to place stickers on a canvas (clear them anytime)  
- 💬 **Floating Bubble** — a personal message from me to Hanni (draggable)  
- 📱 **Fully Responsive** — optimized for both desktop and mobile  

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3** — custom properties, flexbox, grid, animations  
- **JavaScript (Vanilla)** — no frameworks, pure DOM manipulation  
- **LocalStorage** — for saving theme preferences and high scores
